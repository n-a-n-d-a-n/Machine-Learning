# Machine-Learning
Machine Learning (ML) is a subset of Artificial Intelligence (AI) that teaches computers to learn from data and improve at tasks over time, without being explicitly programmed for every scenario, by using algorithms to find patterns and make decisions or predictions. It's the engine behind personalized recommendations, spam filters, self-driving cars, and generative AI, allowing systems to adapt and become more accurate as they process more information. Key types include supervised (labeled data), unsupervised (finding patterns in unlabeled data), and reinforcement learning (learning through rewards).
